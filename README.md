
## webrtc点对点呼叫视频会议

### 简介
  结合Electron构建桌面应用，信令为websocket;
  
  #### 功能：
    呼叫远端用户；
    选择本地媒体设备；
    选择不同分辨率；
    摄像头/麦克风开关切换；
    流媒体信息获取；

## 如何使用
```bash
# Clone this repository
git clone https://gitee.com/silence_doubleL/webrtc-p2p-demo.git
# Go into the repository
cd webrtc-p2p-demo
# Install dependencies
npm install
# Run the app
electron .
```
